# Markdown to HTML converter

This script converts markdown files to html files.

## How to use

Install dependencies

```
npm install
```

Convert

```
node index.js convert --md ./README.md --html ./README.html
```

Where --md=PATH-TO-MARKDOWN-FILE --html=PATH-TO-HTML-FILE
